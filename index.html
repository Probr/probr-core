What is Probr?

An open-source automated testing framework for an ecosystem of community-driven test packs to prove compliance validation in the cloud, on-prem, pre-prod and prod environments alike.

Validate any Resource

Using the Probr SDK, service packs can be built to target any resource, test against any standard, and integrate with other service packs for a seamless runtime!
